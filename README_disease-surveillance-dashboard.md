# Disease Surveillance Dashboard (Power BI)

A multi-page Power BI dashboard for monitoring disease case data — tracking cases, outcomes, demographics, regional spread, trends, and vaccination progress.

## Overview

This dashboard analyzes an epidemiological dataset to track the spread and impact of a disease outbreak, breaking down cases by demographic and geographic factors and monitoring trends over time, including vaccination progress and hospitalization rates.

## Pages

- **Overview** — high-level summary of total cases, active cases, recovered, critical, and deaths, with breakdowns by gender and age group
- **By age group** — case distribution and trends across age groups
- **By region** — geographic spread of cases with map visualization and detailed table
- **Trends** — case trends over time using line and stacked area charts
- **Vaccination** — vaccination progress metrics and breakdowns
- **Alerts page** — key indicators highlighted for monitoring purposes

## Key Metrics Tracked

- Total cases, active cases, recovered, critical, and deaths
- Hospitalization rate
- Case distribution by gender, age group, and region
- Daily/weekly case trends

## Tools & Techniques

- Microsoft Power BI
- Multi-page report design
- DAX measures for aggregated health metrics
- Visuals: clustered bar chart, donut chart, line chart, stacked area chart, map, table, KPI cards
